<p>En este dashboard se hace uso de distintos paquetes para la visualizacion de graficos interactivos: highcharter, plotly y ggplot; De igual manera se apoya en el universo de paquetes tidyverse. A continuacion de detallan los distintos modulos.</p>
<h2 style="color: #2e6c80;">Motor Portfolio TPL:</h2>
<p>Este modulo muestra el performance de una cartera de Resposabilidad Civil Vehiculos.</p>
<p>Despliega graficos descriptivos con la opci&oacute;n de cambiar la variable a estudiar y muestra indicadores en base a los inputs seleccionados. Realiza una simulacion de frecuencia y severidad de siniestros y despliega graficos de barras condicionados a inputs de retencion de Reaseguro. Muestra tambi&eacute;n una tabla con el detalle por poliza.</p>
<h2 style="color: #2e6c80;">Long-Term IRate Forecast:</h2>
<p>Forecast de la tasa de interes a largo plazo del pais seleccionado, utilizando<br />tecnicas de series de tiempo en data mensual. Todos los modelos son desarrollados utilizando el paquete forecast.</p>
<p>En el panel Overview se observa un Mapa de calor con las tasas agrupadas por Cuatrimestre de cada pa&iacute;s disponible. Se despliega tambien graficos de descomposicion de serie de tiempo STL -Seasonal and Trend decomposition using Loess- asi como graficos de Autocorrelacion y Autocorrelacion Parcial.</p>
<p>Se modela la serie de tiempo en data de entrenamiento utilizando varias tecnicas y <br />se realiza el forecast en la data de prueba, mostrando un cuadro comparativo de los modelos y pronosticando el seleccionado por el usuario.</p>
<h2  style="color: #2e6c80;">IBNR Development Triangles:</h2>
<p>Forecast de reclamos incurridos no reportados (IBNR) por medio de triangulos actuariales, utilizando distintas tecnicas:  Chain Ladder, Mack Chain Ladder y Bootstrap Chain Ladder. Para cada tecnica se muestra graficos de diagnostico y los resultados arrojados. Todos   panel Overview se observa un Mapa delos modelos son desarrollados utilizando el paquete ChainLadder.</p>
<h2 style="color: #2e6c80;">Emergency Monitor:</h2>
<p>Monitoreo del estado del clima en situaciones de emergencia. En el primer panel se visualiza el estado actual de la region seleccionada y el pronostico en horas y dias susesivos utilizando para la recoleccion de data en tiempo real la API DakSky. En el segundo panel se utiliza la API de twitter para desplegar un timeline de cuentas informativas asi como graficar la frecuencia de tweets que contienen las palabras introducidas por el usuario en los ultimos 6-9 dias. Se utiliza el paquete rtweet para recolectar los datos a traves de la API de twitter y para desarrollar los graficos.</p>
